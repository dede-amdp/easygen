‚ùì What is EasyGen?


EasyGen is a automatic generation tool that extracts the comments from the code files you upload and generates documentation automatically!

‚ùì How do I use EasyGen?


It's simple! Just upload the files that contain the code and a documentation file will be automatically generated containing the documentation of all the uploaded files.

‚öôÔ∏è Instructions


To be able to extract the comments from a file, EasyGen needs 2 things:
  • Comment blocks: these comment blocks/ multiline blocks need to contain the info that EasyGen will use;
  • Fields: fields are needed to recognize different parts of the information.
All the fields start with a `@` symbol and a field name, followed by the information you want to write, for example

@name method_name
@brief this is an example of what you can do
@note this is just an example and it is possible to add any other user defined fields
@variables
- a: something
- b: something else


the only required fields is the `@name` field which contains the name of the method that you are describing (or of whatever part of the code is going to be described).
Other special fields are `@brief` that contains a summary of what is described and `@note` which can be used freely to add any information is needed.
Lastly, all the other fields (that you can define freely) will be inserted inside a table.

üë®‚Äçüíª Code Snippets!

Code snippets can be added by using the `@codestart` and `@codeend` fields (always between multiline comment blocks): the `@codestart`, followed by the name of the code snippet, has to be put before the code section that you want to include, the `@codeend` field has to be put at the end of the code section. Within these sections, comments can be inserted as normal to explain the function of the code. There can be multiple code snippets (with different names) and they will considered as part of the comment block that precedes them.
This is an example of how to use it:

/*@codestart name1*/
//this is the code that will be inserted
for(var i = 0; i < 10; i++){
console.log("<3\n");
}
/*@codeend*/
